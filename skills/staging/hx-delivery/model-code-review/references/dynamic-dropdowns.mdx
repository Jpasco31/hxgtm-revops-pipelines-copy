## Objectives

## Parameter tables

- Import the `regions` parameter table from the .zip folder provided on the [introduction](introduction) page.

- This parameter table will be used to generate the dynamic dropdown and contains a list of countries, regions and adjustment values.

- Update the `parameter_tables_schema.json` file.

---

## Data Schema

- Rename the `territory` structure to now be called `region`, remember to update the structure's label.
- Next, update the `value` child of region to point to `regions_dropdown` for the dropdown options:
  - `options_data="../../region_dropdown" options_field="region" optionality="optional" default=None`
- Create a new output List called `region_dropdown`, with one child `region` which will be an output String.
- Create a new output List called `country_dropdown`, with one child, `country` which will be an output String.
- Create a new String node `country` with dynamic dropdown options:
  - `options_data="../country_dropdown" options_field="country" optionality="optional" default=None`

<details>
<summary>Click to view the solution</summary>

```python
"region": hx.Structure(view={"label": "Region"}, children={
    "value": hx.Str(mode="input", default=None, options_data="../../region_dropdown", options_field="region", optionality="optional",view={"label": "Value"}),
    "adjustment": hx.Float(mode="output", view={"label": "Adjustment", "format":{"output": "percent", "mantissa":2}}),
    "comment": hx.Str(mode="input", default="", view={"label": "Comment"}),
    }),

"region_dropdown" : hx.List(mode="output", children={
    "region" :hx.Str(mode="output", view={"label": "Region"})
    }),

"country_dropdown": hx.List(mode="output", children={
    "country":hx.Str(mode="output", view={"label": "Country"})
    }),

"country" : hx.Str(mode="input", options_data="../country_dropdown", options_field="country", optionality="optional", default=None, view={"label" : "Country"}),
```

## Rating

- First, read in the new regions parameter table
- Set `hxd.country_dropdown` to the unique list of countries in the regions parameter table, using set().
  > **Note:** remember to sort the list, using sorted(), to ensure your code remains deterministic. Set will return an unordered result.
- Filter the parameter table for the regions associated with the users selected country. Assign the result to the `region_dropdown` list.
- Next, an IF Statement is needed to ensure that a valid region is selected, meaning that the code so far should be:

```python
    region_table = hx.params.regions
    hxd.country_dropdown = sorted(set(region_table["country"]))
    filtered_regions = region_table[region_table["country"] == hxd.country]
    hxd.region_dropdown = filtered_regions["region"]

    if hxd.region.value in filtered_regions.values:
        hxd.region.adjustment = filtered_regions[filtered_regions["region"] == hxd.region.value]["adjustment"].iloc[0]
    else:
        hxd.region.adjustment = 0
        hx.errors.validation("Please select a valid region")
```

- When calculating the total adjustment (`hxd.total_adjustments`), `hxd.territory.adjustment` needs to be changed to `hxd.region.adjustment`.

```python
hxd.total_adjustments = (1+hxd.region.adjustment)*(1+hxd.loss_history.adjustment)*(1+hxd.exposure.adjustment)-1
```

- Remove the two lines associated with Territory Adjustment.
