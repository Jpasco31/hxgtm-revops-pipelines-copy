## Background information

- Async tasks are used for long-running and non-deterministic elements of your pricing workflow. These can include API calls, SQL fetches/updates, simulations, setting up tables, reading files, calling machine learning modules plus many more!

- This chapter will demonstrate how to fetch data from an external source using an API.

- The following API fetch requires authentication via an API key. A free API key can be generated by following the instructions on the [Companies House website](https://developer.company-information.service.gov.uk/get-started/)

- In the example below, the API key is stored in a secret called `companies_house_key`, be sure to give your model permissions to access the secret.

> Hint: Information on secrets and how to use them can be found [here](/platform/developer/model-development-process/models/managing-model-secrets).

## Updating tasks.py

Later in this chapter when updating the view you will create a button that will be used to run the API call. Within `tasks.py` you will find the code to complete the API call which runs when the button is pressed.

Copy and paste the API Call provided below into `tasks.py`.

<details>
<summary>Click to view API code</summary>

```python
import hx
import requests

@hx.task
def get_companies_house(hxd, progress):

    # Read API key from secrets & 'company number' from hx Renew
    api_key = hx.secrets.companies_house_key
    company_no = hxd.companies_house.company_id

    # Validation to ensure a company number has been entered
    if company_no == "":
        hxd.companies_house.fetch_status = "No Company Number selected"
        return

    # Make fetch from Companies House
    request_url = "https://api.companieshouse.gov.uk/company/%s" %company_no
    hxd.companies_house.fetch_status = "Fetching from Companies House..."
    progress.update(0)
    try:
        response = requests.get(request_url, auth=(api_key, ""))
    except requests.RequestException:
        hxd.companies_house.fetch_status = "Unable to connect to Companies House API"
        return
    if response.status_code != 200:
        print("Error: Unexpected status code: " + str(response.status_code))

        if response.status_code == 404:
            hxd.companies_house.fetch_status = "Company profile not found"
        elif response.status_code == 400:
            hxd.companies_house.fetch_status = "Bad Request"
        else:
            hxd.companies_house.fetch_status = "Failed - unexpected response from server (status code: " + str(response.status_code) + ")"

        return

    hxd.companies_house.fetch_status = "Fetch successful"

    # The returned JSON file is read into 'result'
    result = response.json()

    # Extract address details
    address = result.get("registered_office_address", None)
    if address:
        hxd.companies_house.address_line_1 = address.get("address_line_1","")
        hxd.companies_house.address_line_2 = address.get("address_line_2","")
        hxd.companies_house.locality = address.get("locality","")
        hxd.companies_house.postal_code = address.get("postal_code","")

    # Extract basic company details
    hxd.companies_house.company_name = result.get("company_name","")
    hxd.companies_house.company_status = result.get("company_status_detail","")
    hxd.companies_house.creation_date = result.get("date_of_creation","")

    # Find out whether the company has any overdue accounts, charges or insolvency history
    hxd.companies_house.accounts_overdue = result.get("accounts", None).get("overdue", None)
    hxd.companies_house.has_charges = result.get("has_charges", None)
    hxd.companies_house.has_insolvency_history = result.get("has_insolvency_history", None)

    progress.update()

```

</details>

## Updating the data schema

In your data schema, create a new structure called `companies_house` with the following child nodes:

| Node                     | Data Type | Mode   |
| ------------------------ | --------- | ------ |
| `fetch_status`           | str       | output |
| `company_name`           | str       | output |
| `company_status`         | str       | output |
| `creation_date`          | str       | output |
| `address_line_1`         | str       | output |
| `address_line_2`         | str       | output |
| `locality`               | str       | output |
| `postal_code`            | str       | output |
| `accounts_overdue`       | bool      | input  |
| `has_charges`            | bool      | input  |
| `has_insolvency_history` | bool      | input  |

To allow the task to assign a value to these variables, also include an additional property; `async_output=”get_companies_house”`

Add an addition child node to `companies_house` called `company_id`:

- company_id (str, input)

Add the property `aysnc_input=["get_companies_house"]` to `company_id` to allow the variable to be used as an input to the task.

<details>
<summary>Click to view data schema solution</summary>

```python
   "companies_house" : hx.Structure(children={
            "fetch_status" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Fetch Status"}),
            "company_name" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Company Name"}),
            "company_status" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Company Status"}),
            "creation_date" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Date of Creation"}),
            "address_line_1" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Address Line 1"}),
            "address_line_2" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Address Line 2"}),
            "locality" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Locality"}),
            "postal_code" : hx.Str(mode="output", async_output=["get_companies_house"], view={"label":"Post Code"}),
            "accounts_overdue": hx.Bool(mode="input", default=None, async_output=["get_companies_house"], optionality="optional", view={"label": "Accounts Overdue"}),
            "has_charges" : hx.Bool(mode="input", default=None, async_output=["get_companies_house"], optionality="optional", view={"label": "Charges"}),
            "has_insolvency_history": hx.Bool(mode="input", default=None, async_output=["get_companies_house"], optionality="optional", view={"label": "Insolvency History"}),
            "company_id" : hx.Str(mode="input", async_input=["get_companies_house"], default="", view={"label":"Company Number","format":{"thousandSeparated":False, "mantissa":0}}),
        }),
```
