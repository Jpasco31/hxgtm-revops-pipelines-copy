---
title: "Running Model Tools"
---

The code editor contains some tools to help you when developing your models. These tools are accessed and run in the same way:

#### Show the Debug Panel

1. From the side menu select the "Debug" icon.

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/tools_debugger.png").default}
  width="400"
/>

2. This will expand the debug panel.

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/tools_debug_panel.png").default}
  width="400"
/>

#### Show the Debug Output Panel

1. From the View menu, select "Debug Console"

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/tools_debug_console.png").default}
  width="400"
/>

This will add the "Debug" panel to the bottom bar

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/tools_view_debug_console.png").default}
  width="800"
/>

#### Run the Tool

1. In the "Debug" panel, select the required tool from the dropdown

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/tools_select_test.png").default}
  width="400"
/>

2. Click the "Run" button to run the selected tool

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/tools_debugging.png").default}
  width="400"
/>
