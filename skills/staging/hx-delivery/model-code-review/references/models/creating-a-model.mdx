---
title: "Creating a Model"
---

To create a new Model:

1. Log in to the Developer Dashboard (see [Developer Dashboard](/platform/developer/model-development-process/developer-portal) for details).

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/developer_dashboard.png").default} 
  width="1000"
/>

2. Use the left menu to navigate to the models screen, and click the 'Create Model' button.

<img
  src={require("../../assets/ModelDevelopmentProcess/Models/developer_view_models.png").default} 
  width="1000"
/>

3. Fill in the details of the new model

The Responsible Developer and Responsible Underwriter fields are provided so that you can keep track of your model usage, in order to allow you to easily report on your Model landscape.
