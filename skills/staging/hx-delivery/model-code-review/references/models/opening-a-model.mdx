## Opening a Model for Editing

1. Log in to the Developer Dashboard

2. Select 'Models' from the left menu, here you will see a list of all the models that you have access to. Here you can use the search bar to find the model you wish to open.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_model_list.png').default}
   width="1000"
/>

3. Click on the model that you wish to open, this will present you with an overview of the model.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_model_preview.png').default}
   width="1000"
/>


4. Hover over the branch that you wish to open and the 'Create Draft Version' button will appear.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_create_draft.png').default}
   width="1000"
/>

5. The draft version will then be available for editing and you will see an overview of the version, including any User Libraries and Python Packages that are installed.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_draft_version.png').default}
   width="1000"
/>

6. From here you can open the code editor.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_code_editor.png').default}
   width="1000"
/>

7. The code editor will open in a new browser tab.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_code_editor_view.png').default}
   width="1000"
/>

Note: _that you may only have one Model open for editing at any time_

## Opening a Model in Read-Only Mode

1. Log in to the Developer Dashboard

2. Select any model, and find the branch containing the version which you would like to view, in this example the version exists within the branch called 'Main'.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_read_only_branch.png').default}
   width="1000"
/>

3. Click on the model branch. This will bring up the list of versions for this model.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_read_only_versions.png').default}
   width="1000"
/>

4. Click on the 'More Actions' button, then select 'Open read only'.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_read_only.png').default}
   width="1000"
/>

5. The selected model version will appear in at the top of page. Click on this to see an overview of the model version.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_view_read_only.png').default}
   width="1000"
/>

6. From here you can open the code editor or preview the model. You will see the model files the code editor in the `read-only` folder inside the code editor.

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/read_only_code_editor.png').default}
   width="400"
/>

## Viewing an Open Model

If you have opened a model, you can bring up a live preview of the model. If the model is open for editing, any changes you make to the model code in the code editor will automatically update the preview.

1. Open a model (following the instructions above)

2. Click on the 'Preview' button

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_code_editor.png').default}
   width="1000"
/>

3. The live preview will be opened in a new tab of the browser

<img
   src={require('../../assets/ModelDevelopmentProcess/Models/open_preview_model.png').default}
   width="1000"
/>
