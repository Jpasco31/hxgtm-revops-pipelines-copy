## Parameter tables

- Name your parameter tables and columns clearly and concisely, avoiding abbreviations.
- Define column names in lower case, with underscore replacing spaces.
- No spaces between columns.
- All elements in a single column must be of the same type.
- If there is a string which contains a comma within the parameter table, put them in quotation marks to avoid getting an error.

```json
product,adjustment
Offshore Energy,0.1
"Onshore Energy, Oil & Gas",0.4
"Onshore Energy, Power Gen",0.3
```

## Parameter table schema

- Match parameter table and column names exactly.
- Define parameter tables in alphabetical order.
