# Python Documentation

The Python documentation serves as the official, comprehensive guide to the Python programming language. It is developed and maintained by the Python Software Foundation; it is a collective effort by the Python community to provide extensive documentation for Python users. It contains detailed information on language syntax, standard libraries, best practices and much more. It is the single most reliable reference for Python developers.

In this chapter, we will discuss the different sections of the Python documentation are and how you can use each of them. We are only going to focus on the three most important parts of the documentation, which are:

- Tutorial
- Standard Library Reference
- Language Reference

<img src={require("../assets/python_doc_homepage.gif").default} width="800" />

On the homepage of the [Python documentation](https://docs.python.org/3/), you can browse the different sections, select a Python version (see left sidebar), or look for keywords in the documentation with the search bar (top right corner).

> **_Note:_** We would like to iterate again that the Python documentation linked above is the most reliable source of reference for Python. If you happen to encounter contradictions with ChatGPT and Stack Overflow, always rely on the Python documentation over the other tools.

## Tutorial

Created to introduce newcomers to the language, it features self-contained examples and step-by-step guides, offering hands-on experience with Python's fundamental concepts and features. Consider this a beginner’s manual with step-by-step guides to understand the basics of Python.
Click [here](https://docs.python.org/3/tutorial/index.html) to explore the Python Tutorial.

<img src={require("../assets/python_doc_tutorial.gif").default} width="800" />

### How to use:

- You are able to work through this section sequentially, or by directly clicking on the topics you are interested in.
- There is a brief explanation on how everything works and possible limitations.
- You can also review one or more code examples to see a real application.

## Standard Library Reference

On the documentation homepage it says “keep this under your pillow” and there is a good reason for that. The Python Standard Library documentation is a guide to the built-in modules and packages that come with Python. It lists all functionalities and, if applicable, all attributes, arguments and expected outputs. The purpose is to provide you with enough information about a module or package so you can use it in your own code.

Click [here](https://docs.python.org/3/library/index.html) to explore the Python Standard Library Reference.

<img src={require("../assets/python_doc_std_lib.gif").default} width="800" />

### How to use:

- We recommend you use the search bar to look up anything you want to learn more about.
- You can also browse through the table of contents to find the overarching topic that you are interested in.
- You can look up a short definition of every function and what it can do.

> **_Note:_** The Python Standard Library documentation does not cover any functionalities outside of the standard library. Meaning, if you are interested in modules and packages of other libraries, you must go to their respective documentation. Examples are [pandas](https://pandas.pydata.org/docs/), [NumPy](https://numpy.org/doc/), [MATLAB](https://www.mathworks.com/help/matlab/), [Plotly](https://plotly.com/python/), [Matplotlib](https://matplotlib.org/stable/index.html), [scikit-learn](https://scikit-learn.org/stable/), [Keras](https://keras.io/) and [Tensorflow](https://www.tensorflow.org/).

## Language Reference

Described as your guide to "speaking Python fluently," the Language Reference in the Python documentation provides an in-depth exploration of Python's vocabulary and inner workings. It goes beyond discussing code directly, focusing on the meaning and intricacies of the language. Its purpose is to offer detailed insights into Python's syntax, semantics, and core features, making it an invaluable resource for those seeking a deeper understanding of the language.

Click [here](https://docs.python.org/3/reference/index.html) to explore the Python Language Reference.

<img src={require("../assets/python_doc_lang_lib.gif").default} width="800" />

### How to use:

- Study Python topics in-depth and read through the chapters you are interested in.
- Reading the language reference is more time-consuming, so think about when it makes sense to dive deeper into certain topics.

> **_Note:_** The Language Reference specifically covers Python's language specifications and does not delve into code implementation.

## Python Tutor

The Python Tutor visualization is a tool that offers a step-by-step, visual representation of code execution. Its primary purpose is to help users understand the flow and behavior of Python code by visually illustrating how variables change and interact during program execution. This interactive visualization tool serves as a helpful aid for beginners in grasping programming concepts and troubleshooting code.

Click [here](https://pythontutor.com/) to visualize your code with the Python Tutor.

<img src={require("../assets/python_tutor.gif").default} width="800" />

### How to use:

1. Go to the Python Tutor homepage and click “Python”
2. Now below the code box click “Edit code and get AI help”
3. Copy and paste your code into the code box and hit “Visualize Execution”
4. Navigate through the execution as you like clicking the buttons below the code box
5. On the right-hand side you can see all outputs of the separate executions
6. A generative AI will try to answer your questions
